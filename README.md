# Taskflow-AI

TaskFlow AI is an intelligent project management dashboard that leverages AI to enhance developer productivity. It connects with GitHub repositories to automatically track progress and provides smart suggestions for task prioritization and project optimization.

This app is for my 100 hours coding on github for Hack Club Neighborhood!

## Core Features

### GitHub Integration
- Connect to repositories
- Auto-track commits and PRs to update task status
- Visualize contribution metrics

### AI-Powered Features
- Task prioritization suggestions
- Automated status updates based on commit messages
- Sprint planning recommendations
- Deadline risk assessment
- Code review suggestions

### Dashboard Elements
- Kanban board with drag-and-drop functionality
- Burndown charts and velocity metrics
- Team member workload visualization
- Time tracking and estimation accuracy
- Project health indicators

## Tech Stack
- Next.js with TypeScript
- Tailwind CSS for styling
- Prisma with PostgreSQL for data management
- Firebase for authentication
- GitHub API for repository integration
- Google AI Studio API for intelligent suggestions
